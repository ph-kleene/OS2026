# OS2026

这是2026年操作系统（Operating System）课程的学习记录、课后作业与实验代码库。

## 📁 目录结构

本仓库包含了操作系统课程的相关实践内容：

### 🔬 实验 (Labs)
- **[lab1](./lab1)**: 实验一 — 引导扇区与实模式编程
- **[lab2](./lab2)**: 实验二 — 保护模式与系统调用
- **[lab3](./lab3)**: 实验三 — 多进程管理与调度
- **[lab4](./lab4)**: 实验四 — 进程间通信与同步（信号量、Wait/Exit、时间片轮转调度）

### 📝 作业 (Homeworks)
- **[homework1](./homework1)**: 作业一
- **[homework2](./homework2)**: 作业二
- **[homework3](./homework3)**: 作业三 — 读写锁与环形缓冲区

## 🛠️ 环境依赖 (Prerequisites)

- Linux 操作系统
- GCC / G++ 编译器
- Make 构建工具
- 相关的系统环境配置视具体实验/作业中的说明而定。
